MemoriaR's Privacy Policy

This privacy policy applies to the MemoriaR app (hereby referred to as the "Application") for mobile devices that was created by Natsuki Harai (hereby referred to as the "Service Provider"). The Application is provided as a Freemium service and is intended for use "AS IS".

Information Collection and Use

The Application does not collect, transmit, or process personal information or usage data on servers managed by the Service Provider.

The Application stores limited data only for the purpose of providing its core functionality. The following information is stored locally on the user’s device and synchronized using Apple’s iCloud service (CloudKit):

  • Content identifiers used to manage items created within the Application
  • Internal identifiers used solely for application-level data management

This data is used exclusively to enable application functionality, device backup, and synchronization across the user’s own devices.

CloudKit (iCloud)

The Application uses Apple’s CloudKit (iCloud) to store and synchronize application data.

  • Data is stored in the user’s own iCloud account
  • The Service Provider does not have access to, and cannot view, retrieve, or analyze user data stored in CloudKit
  • Apple acts as the infrastructure provider for CloudKit

The handling of data within CloudKit is governed by Apple’s own privacy policies.

Information Not Collected

The Application does not collect, track, or transmit any of the following information:

  • Names, email addresses, contact details, or other personal information
  • IP addresses
  • Application usage history, page views, or time spent
  • Location or precise geographic information
  • Advertising identifiers or tracking data
  • Analytics, marketing, or profiling data

Third-Party Access

The Application does not use third-party analytics, advertising, or marketing services.
User data is not shared, sold, or disclosed to third parties.

Apple’s CloudKit service is used solely as a data storage and synchronization mechanism and does not grant the Service Provider access to user data.

Data Retention and Deletion

  • Data stored on the device is removed when the Application is uninstalled
  • Data stored in iCloud can be deleted by the user through their iCloud settings

The Service Provider does not retain copies of user data.

Children

The Application may be used by users of any age.
The Application does not knowingly collect personal information from children or adults.

Security

This Privacy Policy may be updated from time to time for any reason. Any changes will be reflected by updating this page with the revised Privacy Policy. Continued use of the Application constitutes acceptance of the updated policy.

This privacy policy is effective as of January 1st 2026.

Your Consent

By using the Application, you consent to this Privacy Policy.

Contact Us

If you have any questions regarding privacy while using the Application, please contact the Service Provider:

  • Email: superhahnah@gmail.com
